mpyw先生の引用です。

---

実行コマンド

`a=rX,u+wi`


```
a: 全員のパーミッションを
=: ゼロにクリアして次のように設定する
r: 読み込み権限を付加 [+4]
X: ディレクトリのみ実行権限を付加 [+1]
```

```
u: 所有者のパーミッションを
+: 次のように上書きする
w: 書き込み権限を付加 [+2]]]]
```

結果

```
ファイル: 644
ディレクトリ: 755
```
